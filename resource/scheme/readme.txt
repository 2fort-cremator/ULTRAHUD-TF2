This is clientscheme. Clientscheme is where "larger" changes and more important stuff tends to be stored.

note: changes to clientscheme require a full game restart to take effect.

DOCS:

borders - controls hud borders (eg. backpack items, menu popups, etc...)

fonts - controls what fonts the HUD has
fonts_bold - does nothing
icons - controls icon-specific font files, do not touch

colors - controls predefined colors

xhairs / xhairs_outline - controls hud crosshairs
